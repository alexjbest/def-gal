Example 2.7.

Let \(E/\QQ\) be an elliptic curve

\begin{equation*} G_\QQ \acts E\lb p\rb \simeq \ZZ/p \oplus \ZZ/p \end{equation*}
\begin{equation*} \rho _{E,p} \colon G_\QQ \to \Aut(E\lb p\rb ) \simeq \GL_2(\FF_p) \end{equation*}
\begin{equation*} \rho _{E,p^n} \colon G_\QQ \to \Aut(E\lb p^n\rb ) \simeq \GL_2(\ZZ/p^n) \end{equation*}
\begin{equation*} \rho _{E,p^\infty} \colon G_\QQ \to \Aut(E\lb p^\infty\rb ) \simeq \GL_2(\ZZ/p^\infty)\text{.} \end{equation*}
