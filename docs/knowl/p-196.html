The proof of the claim is as follows: Let

\begin{equation*} \tilde \rho \end{equation*}

be \(\rho ^{univ}\) mod \(p\)

\begin{equation*} \tilde \rho\colon G \to \GL_d(R_{\bar\rho }/pR_{\bar\rho }) \end{equation*}
\begin{equation*} \sheaf O (\tilde \rho ) \in H^2(G, \ad \bar \rho ) \otimes J/\ideal m J \end{equation*}
\begin{equation*} \Hom(J/\ideal m J,k) \xrightarrow\alpha H^2(G, \ad \bar\rho ) \end{equation*}
\begin{equation*} f\mapsto (1\otimes f)(\sheaf O(\tilde \rho )) \end{equation*}

\(f\ne 0\) so that

\begin{equation*} \alpha (f) = 0 \end{equation*}
\begin{equation*} f\colon J/\ideal m J \to k \end{equation*}
\begin{equation*} 0 \to (J/\ideal m J)/ \ker f \to ((k \lb \lb X_1, \ldots, X_{d_1}\rb \rb )/\ideal m J)/ \ker f \to R_{\bar \rho }/ p R_{\bar\rho } \to 0\text{.} \end{equation*}

But this is

\begin{equation*} 0 \to k \to R'\to R_{\bar\rho }/pR_{\bar\rho } \to 0\text{.} \end{equation*}

Obstruction class is \(f (\sheaf O (\tilde \rho )) = 0\text{.}\) So there exists a deformation of \(\tilde \rho \) to \(R'\text{.}\) But \(\characteristic R' = p\) as

\begin{equation*} \tilde \rho \end{equation*}

is the universal deformation of \(\bar\rho \) to a characteristic \(p\) ring.
