We need to know that \(p^n \ne 0\) in \(R_{\bar \rho }\text{.}\) Suffices to show that

\begin{equation*} D_{\bar \rho }(W(k)) \ne \emptyset\text{.} \end{equation*}

Having \(H^2 = 0\) implies we can lift small homomorphisms

\begin{equation*} k \simeq W(k) / p \leftarrow W(k)/p^2 \leftarrow \cdots \end{equation*}

so we can lift to \(W(k)\text{.}\)
