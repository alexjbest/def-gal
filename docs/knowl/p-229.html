A funny observation is that

\begin{equation*} (V_A, i_A) \simeq (V_A, i'_A)\text{.} \end{equation*}
\begin{equation*} (i_A')\inv \circ i_A \in \End(V_A \otimes k ) = \End(\bar \rho ) \simeq k \end{equation*}

so we are just scaling by \(c \in k^*\text{.}\) But we can take \(\phi = \lb c\rb \text{.}\)
