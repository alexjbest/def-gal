Note that when \(k_1,k_2\in \ZZ\) with

\begin{equation*} k_1\equiv k_2 \pmod{p-1} \end{equation*}

then

\begin{equation*} \epsilon _p^{k_1} \equiv \epsilon _p^{k_2} \pmod{p} \end{equation*}

moreover if

\begin{equation*} k_1\equiv k_2 \pmod{(p-1)p^N} \end{equation*}

we havee

\begin{equation*} \epsilon _p^{k_1} \equiv \epsilon _p^{k_2} \pmod{p^{N+1}}\text{.} \end{equation*}
