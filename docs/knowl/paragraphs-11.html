Tangent spaces and deformations.

Let

\begin{equation*} k\lb \epsilon \rb = k\lb x\rb /(x^2) \end{equation*}

then \(\epsilon = \bar x\) and we have \(\epsilon ^2 =0\text{.}\) These are known as the dual numbers.

Proposition 2.41.
\begin{equation*} \Hom_{\mathcal C} (A, k\lb \epsilon \rb ) \simeq t_A \end{equation*}

for \(A \in \mathcal C\text{.}\)

Proof.

Take \(A = R_{\bar \rho }\) then the proposition implies

\begin{equation*} D_{\bar \rho } (k\lb \epsilon \rb ) \simeq t_{R_{\bar\rho }} \end{equation*}

to deform

\begin{equation*} \bar \rho \end{equation*}

to \(k\lb \epsilon \rb \)

\begin{equation*} \rho _\epsilon(g) = \bar \rho (g) + \epsilon \phi (g) \end{equation*}

for some

\begin{equation*} g\mapsto \phi (g) \in M_d(k) \end{equation*}

with

\begin{equation*} \xymatrix{ G \ar[r]^{\rho _\epsilon } \ar[dr]_{\bar\rho } & \GL_d(k\lb \epsilon \rb )\ar[d]\\ & \GL_d(k) }\text{.} \end{equation*}

We know

\begin{equation*} \rho _\epsilon (gh) = \rho _\epsilon (g) \rho _\epsilon (h) \end{equation*}
\begin{equation*} \bar \rho (gh) + \epsilon \phi (gh) = (\bar \rho (g) + \epsilon \phi (g))(\bar \rho (h) + \epsilon \phi (h)) \end{equation*}
\begin{equation*} = \bar \rho (g)\bar \rho (h) + \epsilon ( \phi (g)\bar \rho (h) +\bar \rho (g) \phi (h)) \end{equation*}

so

\begin{equation*} \phi (gh) = \phi (g) \bar \rho (h)+\bar \rho (g) \phi (h) \end{equation*}

scale on the right by

\begin{equation*} \bar \rho (h)\inv \bar \rho (g)\inv = \bar \rho (gh)\inv \text{.} \end{equation*}
\begin{equation*} \phi (gh) \bar \rho (gh)\inv = \phi (g) \bar \rho (g)\inv +\bar \rho (g) \phi (h) \bar \rho (h)\inv \bar \rho (g)\inv \end{equation*}

set

\begin{equation*} \psi (g) = \phi (g) \bar \rho (g)\inv \end{equation*}

so

\begin{equation*} \psi (gh) = \psi (g) + \bar \rho ( g) \psi (h) \bar \rho (g)\inv \end{equation*}
\begin{equation*} \psi (h) \in M_d(k) \simeq \End(\bar \rho )\acts G \end{equation*}
\begin{equation*} \End (\bar \rho ) = \Hom(V_{\bar \rho },V_{\bar \rho }) = \ad(\bar \rho ) \end{equation*}

this is the representation where

\begin{equation*} g \bullet M = \bar \rho (g) M \bar \rho (g)\inv \text{.} \end{equation*}
\begin{equation*} \psi \in Z^1 (G, \ad(\bar \rho )) \end{equation*}

so we get a map

\begin{equation*} \{ \text{lifts of }\bar \rho \} \to Z^1(G, \ad \bar \rho )\text{.} \end{equation*}

\begin{equation*} 1+ pM_d(\FF_p) \simeq M_d(\FF_p) \simeq \End(\bar \rho )\text{.} \end{equation*}
Claim 2.46.
\begin{equation*} \phi (g,h) \in Z^2(G, \ad \bar \rho ) \end{equation*}

The condition to be such a cocycle is

\begin{equation*} a \bullet \phi (b,c) - \phi (ab,c) + \phi (a,bc) - \phi (a,b) = 0\text{.} \end{equation*}

So we do a long calculation, using the fact that the kernel is abelian.

\begin{equation*} A_1 \xtwoheadrightarrow {\pi } A_0 \end{equation*}
\begin{equation*} I = \ker \pi \end{equation*}

s.t.

\begin{equation*} \ideal m_{A_1} I = 0 \end{equation*}

\begin{equation*} \phi (g,h) = \rho _1(gh) \rho _1(h)\inv \rho _1(g)\inv \equiv 1 \pmod I \end{equation*}

so in

\begin{equation*} 1 + M_d(I) \simeq 1 + M_d(k) \otimes I \end{equation*}
\begin{equation*} \simeq \ad \bar \rho \otimes I\text{.} \end{equation*}
