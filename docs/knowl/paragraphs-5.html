Framed deformations.
\begin{equation*} D_{\bar \rho }^{\square} (A) = \{ \rho \colon G\to \GL_d(A)\text{ lifting }\bar \rho \} \end{equation*}

we will prove that

\begin{equation*} D_{\bar \rho }^{\square} \end{equation*}

is in fact representable.

\begin{equation*} \bar \rho \colon G \to \GL_d(k) \end{equation*}

we build

\begin{equation*} R_{\bar \rho }^{\square, univ} \end{equation*}

and

\begin{equation*} \rho _{\bar \rho }^{\square, univ} \colon G \to \GL_d(R_{\bar \rho }^{\square, univ})\text{.} \end{equation*}

Let \(\{ g_\alpha \}_\alpha \) be a generating set of \(G\text{.}\) Make formal variables \(X_{ij}^\alpha \) for all \(\alpha \text{,}\) \(1 \le i,j \le d\text{.}\)

\begin{equation*} R = W(k) \lb \lb \{X_{ij}^\alpha \}\rb \rb \end{equation*}

try sending

\begin{equation*} g_\alpha \mapsto (X_{ij}^\alpha ) \in M_d(R) \end{equation*}

doesn't land in \(\GL_d\text{.}\)

So “add on \(\bar \rho \)”.

\begin{equation*} k\to W(k) \end{equation*}
\begin{equation*} x\mapsto [x] \end{equation*}

so we get

\begin{equation*} \GL_d(k) \xrightarrow{\lb \cdot \rb } \GL_d(W(k)) \end{equation*}
\begin{equation*} \lb (a_{ij})_{ij} \rb \mapsto (\lb a_{ij}\rb )_{ij} \end{equation*}

now send

\begin{equation*} g_\alpha \mapsto (X_{ij}^\alpha ) + \lb \bar \rho (g_\alpha ) \rb \in \GL_d(R) \end{equation*}

this does not give a map

\begin{equation*} G \to \GL_d(R) \end{equation*}

because of relations between \(g_\alpha \text{.}\)

Example 2.30.

Take the relation

\begin{equation*} g_1 g_2\inv = e \end{equation*}

need

\begin{equation*} ((X_{ij}^1) + \lb \bar\rho (g_1)\rb )((X_{ij}^2) + \lb \bar\rho (g_2)\rb )\inv - 1 = 0 \end{equation*}

this gives \(d^2\) equations in \(R\text{.}\)

Can do this in general. In fact the LHS is in \(\ideal m_R\text{.}\) So if \(I\) is the ideal of all relations, then \(I\subseteq \ideal m_R\text{.}\) And let

\begin{equation*} R_{\bar \rho }^{univ,\square} = R/I\text{.} \end{equation*}
\begin{equation*} \rho \colon G \to \GL_d(R/I) \end{equation*}
\begin{equation*} g_\alpha \mapsto (X_{ij}^\alpha ) + \lb \bar\rho (g_\alpha )\rb \pmod I \end{equation*}

is a well-defined homomorphism. And it is universal: If

\begin{equation*} \rho _A \colon G \to \GL_d (A) \end{equation*}
\begin{equation*} R\xrightarrow\pi A \end{equation*}
\begin{equation*} (X_{ij}^\alpha )_{ij} \mapsto (\rho _A(g_\alpha ) - \lb \bar\rho (g_\alpha )\rb )_{ij} \end{equation*}

then \(\pi \) kills \(I\) (exercise).

\begin{equation*} R/I \xrightarrow{\bar \pi } A\text{.} \end{equation*}
Exercise 2.31.

There are two problems with this, it is not necessary that the universal deformation we have constructed be continuous, or that the universal deformation ring be continuous.

The root of these problems is that there are potentially too many \(g_\alpha \text{.}\) introduce a condition \(\Phi _p\) that for all open finite index

\begin{equation*} G_0 \subseteq G \end{equation*}

we have

\begin{equation*} \Hom(G_0, \FF_p) \end{equation*}

is finite.
