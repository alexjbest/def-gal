Space of universal framed deformations.
\begin{equation*} \Hom (R_{\bar \rho }^\square, A) \end{equation*}

For unframed deformations it can be the casee that we even have that

\begin{equation*} R_{\bar \rho } \simeq \ZZ_p \lb \lb X_1, X_2, X_3\rb \rb \end{equation*}
\begin{equation*} A = \ZZ_p \end{equation*}
\begin{equation*} D_{\bar \rho } (\ZZ_p) = \Hom_{\mathcal C}(\ZZ_p\lb \lb X_1,X_2,X_3\rb \rb , \ZZ_p ) \simeq (p\ZZ_p)^3\text{.} \end{equation*}

To prove this space exists for unframed deformations, we have 3 options

  1. We should have a bigger space of framed deformations in general, which should quotient down to that of unframed deformations, by forgetting the framing. Then we could construct \(R_{\bar\rho }\) using this quotient. Kisin's notes chapter 3.
  2. Pseudo-deformations (for later).
  3. Schlessinger's criterion.
