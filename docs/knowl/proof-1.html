Proof
\begin{equation*} \Hom_{\mathcal C} (A, k\lb \epsilon \rb ) \simeq \Hom_{\mathcal C} (A/\ideal m_A^2 + pA, k\lb \epsilon \rb ) \end{equation*}

since

\begin{equation*} (\ideal m_{k\lb \epsilon \rb })^2 = 0 \end{equation*}

and

\begin{equation*} p = 0 \in k\lb \epsilon \rb \text{.} \end{equation*}
\begin{equation*} A/\ideal m_A^2 + pA \simeq k \oplus \ideal m_A/\ideal m_A^2 + pA \end{equation*}
\begin{equation*} x\mapsto (\bar x, x - \lb \bar x\rb ) \end{equation*}
\begin{equation*} \lb \bar x\rb + y \mapsfrom (\bar x, y) \end{equation*}

as \(k\)-vector spaces.

\begin{equation*} k\lb \epsilon \rb \simeq k \oplus \epsilon k\text{.} \end{equation*}
\begin{equation*} \Hom_{\mathcal C} (A/\ideal m_A^2 + pA, k\lb \epsilon \rb ) \simeq \Hom_{\mathcal C} (A/\ideal m_A^2 + pA, \epsilon k ) \simeq t_A \end{equation*}

where now we take homomorphisms of \(k\)-vector spaces.
