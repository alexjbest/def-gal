Claim: there exists

\begin{equation*} \Hom(J/\ideal mJ, k) \hookrightarrow H^2(G,\ad \bar\rho ) \end{equation*}

this implies

\begin{equation*} \dim_k (J / \ideal m J) \le d_2 \end{equation*}

so \(J\) has a set of generators of length \(\le d_2\) by Nakayama. So

\begin{equation*} \krulldim(R_{\bar\rho }/pR_{\bar\rho }) \ge d_1 - d_2\text{.} \end{equation*}

Maybe this is Krull's principal ideal theorem.

The proof of the claim is as follows: Let

\begin{equation*} \tilde \rho \end{equation*}

be \(\rho ^{univ}\) mod \(p\)

\begin{equation*} \tilde \rho\colon G \to \GL_d(R_{\bar\rho }/pR_{\bar\rho }) \end{equation*}
\begin{equation*} \sheaf O (\tilde \rho ) \in H^2(G, \ad \bar \rho ) \otimes J/\ideal m J \end{equation*}
\begin{equation*} \Hom(J/\ideal m J,k) \xrightarrow\alpha H^2(G, \ad \bar\rho ) \end{equation*}
\begin{equation*} f\mapsto (1\otimes f)(\sheaf O(\tilde \rho )) \end{equation*}

\(f\ne 0\) so that

\begin{equation*} \alpha (f) = 0 \end{equation*}
\begin{equation*} f\colon J/\ideal m J \to k \end{equation*}
\begin{equation*} 0 \to (J/\ideal m J)/ \ker f \to ((k \lb \lb X_1, \ldots, X_{d_1}\rb \rb )/\ideal m J)/ \ker f \to R_{\bar \rho }/ p R_{\bar\rho } \to 0\text{.} \end{equation*}

But this is

\begin{equation*} 0 \to k \to R'\to R_{\bar\rho }/pR_{\bar\rho } \to 0\text{.} \end{equation*}

Obstruction class is \(f (\sheaf O (\tilde \rho )) = 0\text{.}\) So there exists a deformation of \(\tilde \rho \) to \(R'\text{.}\) But \(\characteristic R' = p\) as

\begin{equation*} \tilde \rho \end{equation*}

is the universal deformation of \(\bar\rho \) to a characteristic \(p\) ring.

So there exists

\begin{equation*} R_{\bar\rho }/p R_{\bar\rho } \to R' \end{equation*}

hence

\begin{equation*} t_{R'} \end{equation*}

has larger dimension than

\begin{equation*} t_{R_{\bar \rho }/pR_{\bar \rho }} \end{equation*}

a contradiction.

For the second part

\begin{equation*} 0 \to J \to k\lb \lb X_1,\ldots, X_{d_1}\rb \rb \to R_{\bar \rho }/ p R_{\bar\rho } \to 0 \end{equation*}

so

\begin{equation*} \dim(J/\ideal mJ) \le d_2 = 0 \end{equation*}

so

\begin{equation*} J/\ideal mJ = 0 \end{equation*}

i.e.

\begin{equation*} J = 0\text{.} \end{equation*}

Hence

\begin{equation*} R_{\bar \rho }/pR_{\bar\rho } \simeq k \lb \lb X_1, \ldots, X_{d_1}\rb \rb \text{.} \end{equation*}

We need to know that \(p^n \ne 0\) in \(R_{\bar \rho }\text{.}\) Suffices to show that

\begin{equation*} D_{\bar \rho }(W(k)) \ne \emptyset\text{.} \end{equation*}

Having \(H^2 = 0\) implies we can lift small homomorphisms

\begin{equation*} k \simeq W(k) / p \leftarrow W(k)/p^2 \leftarrow \cdots \end{equation*}

so we can lift to \(W(k)\text{.}\)
